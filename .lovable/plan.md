

# Fix Translation in Creator Performance Cards

## Problem
The `CreatorPerformanceCard` component has no translation support at all. Three types of untranslated text:

1. **Relevance explanation** -- hardcoded Czech sentences generated by `getRelevanceExplanation()` function (lines 55-93)
2. **Dynamic AI content** -- `content_summary` and `key_insight` are stored in Czech and need the `TranslatedText` component for async AI translation
3. **Minor static labels** -- "No content summary", "No insight available", "No positive/negative topics" fallback texts are in English but the relevance explanation Czech strings need dictionary entries

## Solution

### File: `src/components/reports/CreatorPerformanceCard.tsx`

**1. Add translation imports:**
- Import `useT` from `@/lib/translations`
- Import `TranslatedText` from `@/components/ui/TranslatedText`

**2. Wrap dynamic AI text with `TranslatedText`:**
- Content summary display (line 252): wrap `creator.top_content?.content_summary` in `<TranslatedText text={...} />`
- Key insight display (line 316): wrap `creator.key_insight` in `<TranslatedText text={...} />`

**3. Translate relevance explanation:**
- Wrap the relevance explanation output (line 425) in `<TranslatedText text={...} />`
- This handles the Czech sentences from `getRelevanceExplanation()` via AI translation since they are dynamically composed

**4. Add `useT()` hook** for any remaining static Czech fallback strings if needed.

### File: `src/lib/translations.ts`

Add dictionary entries for the static Czech strings used inside `getRelevanceExplanation` fallback label:
- "značce" -> "brand" (already exists)

No other dictionary additions needed since the relevance text is dynamically composed and best handled by the AI translator.

## Technical Details

The key insight and content summary are AI-generated Czech text stored in the database, so they use the `TranslatedText` component (async AI translation via the edge function with caching). The relevance explanation is also dynamically composed Czech text, so it also gets `TranslatedText` treatment.

### Changes summary:
```text
CreatorPerformanceCard.tsx:
  - line 252: {creator.top_content?.content_summary}
    becomes: <TranslatedText text={creator.top_content.content_summary} />
    
  - line 316: {creator.key_insight || "No insight available"}  
    becomes: <TranslatedText text={creator.key_insight} /> (with fallback)
    
  - line 425: {getRelevanceExplanation(creator, brandName)}
    becomes: <TranslatedText text={getRelevanceExplanation(creator, brandName)} />
```

Only one file needs modification: `src/components/reports/CreatorPerformanceCard.tsx`.
